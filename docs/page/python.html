2-2 Python

Python-pip相关操作命令的集合

命令 介绍
pip install --upgrade pip 自我升级（升级pip到最新版本）
pip install XXX 安装库
pip list 查看当前环境所有已安装的库
pip list --outdated 查看已经过期的库
pip install xxx -U 升级相关库
pip show -f xxx 或者 pip show 查看库安装的路径
pip uninstall xxx 卸载库
